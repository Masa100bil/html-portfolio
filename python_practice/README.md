# Python練習プロジェクト - 数当てゲーム

シンプルな数当てゲームで、Pythonの基本を学べます。

## 実行方法

```bash
cd python_practice
python main.py
```

## 学べる概念

- **関数**: 処理をまとめて再利用可能に
- **条件分岐**: if/elif/else文
- **ループ**: while文で繰り返し処理
- **モジュール**: コードを複数ファイルに分割
- **例外処理**: try/exceptでエラー対応
- **型ヒント**: 引数と戻り値の型を明示
